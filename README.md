
# Ashim Scholars

This is a premier Islamic institution web application built with Next.js, ShadCN UI, and Tailwind CSS.

## How to push to GitHub

To push this project to your repository, run the following commands in your terminal:

```bash
git init
git add .
git commit -m "Initial commit: Ashim Scholars Mobile-First Web App"
git branch -M main
git remote add origin https://github.com/githubnyabintang/ashimScholars.git
git push -u origin main
```

## Features
- **Mobile-First UI**: Native app-like experience on mobile devices.
- **Centric Hero**: Modern, centered landing section.
- **AI Inquiry Assistant**: Integrated Genkit flow to help students draft inquiries.
- **Themed Design**: Professional green and gold aesthetic with a soft glass-morphism feel.
